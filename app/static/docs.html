Documentación de UIResearch

UIResearch es una herramienta profesional para buscar, filtrar y exportar referencias académicas desde la API de Crossref.

🎯 Características principales

  • Búsqueda rápida en millones de artículos académicos
  • Filtros avanzados por fecha, tipo de contenido y más
  • Exportación a CSV y BibTeX
  • Interfaz web moderna y CLI para automatización
  • API REST completa

Instalación

Requisitos previos

  • Python 3.10 o superior
  • pip (gestor de paquetes de Python)

Paso 1: Clonar o descargar el proyecto

git clone https://github.com/tu-usuario/uiresearch.git
cd uiresearch

Paso 2: Instalar dependencias

pip install -r requirements.txt

Paso 3: Configurar variables de entorno

Crea un archivo .env en la raíz del proyecto:

# Crossref Configuration
APP_USER_AGENT=CrossrefSearch/1.0
APP_MAILTO=tu-email@example.com

# Server Configuration
PORT=8000
LOG_LEVEL=INFO

# CORS (comma-separated)
CORS_ORIGINS=http://localhost:8000,http://localhost:3000

# Rate Limiting
RATE_LIMIT_SEARCHES=10/minute
RATE_LIMIT_EXPORTS=5/minute

# Crossref Client
CROSSREF_TIMEOUT=30
MAX_RETRIES=3

Paso 4: Iniciar el servidor

uvicorn app.main:app --host 0.0.0.0 --port 8000

La aplicación estará disponible en http://localhost:8000

Configuración

Variables de entorno

Variable Descripción Valor por defecto
APP_USER_AGENT Identificador de la aplicación CrossrefSearch/1.0
APP_MAILTO Email de contacto (requerido) -
PORT Puerto del servidor 8000
LOG_LEVEL Nivel de logging INFO
CROSSREF_TIMEOUT Timeout para peticiones (segundos) 30
MAX_RETRIES Número máximo de reintentos 3

Uso de la Interfaz Web

Realizar una búsqueda

  1. Ingresa las palabras clave en el campo de búsqueda
  2. Ajusta los filtros según tus necesidades:
    • Fechas: Define el rango temporal
    • Tipo de contenido: Artículo de revista, conferencia o capítulo
    • Ordenación: Por relevancia o fecha de publicación
  3. Haz clic en "Buscar"

Exportar resultados

Una vez obtenidos los resultados, puedes:

  • Exportar a CSV: Haz clic en el botón "Exportar a CSV"
  • Acceder al DOI: Haz clic en el enlace del DOI para ver el artículo completo
  • Ver abstract: Expande la sección "Ver abstract" en cada tarjeta

API REST

GET /search

Busca referencias académicas en Crossref.

Parámetros de consulta

Parámetro Tipo Descripción Por defecto
q string Palabras clave (requerido) -
from_date string Fecha inicio (YYYY-MM-DD) 2023-01-01
until_date string Fecha fin (YYYY-MM-DD) 2025-12-31
content_type string Tipo de contenido journal-article
has_abstract boolean Requiere abstract true
rows integer Resultados por página (1-100) 30
max_results integer Máximo total (1-500) 120
sort string Ordenación (relevance/published) relevance

Ejemplo de petición

curl "http://localhost:8000/search?q=machine+learning&from_date=2023-01-01&rows=10"

Respuesta exitosa (200)

GET /export/csv

Exporta resultados de búsqueda a CSV.

Usa los mismos parámetros que /search.

Ejemplo

curl "http://localhost:8000/export/csv?q=climate+change&rows=50" -o results.csv

GET /export/bibtex

Exporta referencias en formato BibTeX.

Parámetros

  • dois (string): Lista de DOIs separados por comas

Ejemplo

curl "http://localhost:8000/export/bibtex?dois=10.1234/example1,10.1234/example2" -o refs.bib

GET /healthz

Verifica el estado de salud de la aplicación.

curl http://localhost:8000/healthz

GET /metrics

Expone métricas en formato Prometheus.

curl http://localhost:8000/metrics

Ejemplos

Búsqueda simple

curl "http://localhost:8000/search?q=artificial+intelligence"

Búsqueda con filtros

curl "http://localhost:8000/search?q=climate+change&from_date=2020-01-01&until_date=2024-12-31&content_type=journal-article&rows=50"

Exportar a CSV

curl "http://localhost:8000/export/csv?q=quantum+computing&max_results=100" -o quantum.csv

Obtener BibTeX

curl "http://localhost:8000/export/bibtex?dois=10.1038/s41586-019-1666-5" -o paper.bib

Solución de Problemas

Error: "APP_MAILTO is required"

Solución: Asegúrate de configurar la variable APP_MAILTO en tu archivo .env.

Error: "Connection timeout"

Solución: Aumenta el valor de CROSSREF_TIMEOUT en el archivo .env.

Error 429: "Rate limit exceeded"

Solución: Espera unos minutos antes de realizar más búsquedas. Los límites por defecto son:

  • Búsquedas: 10 por minuto
  • Exportaciones: 5 por minuto

No se muestran resultados

Posibles causas:

  • Los filtros son demasiado restrictivos
  • No hay artículos que coincidan con la búsqueda
  • Error de conexión con Crossref

Solución: Intenta con palabras clave más generales o ajusta los filtros.

Ver logs

Los logs se muestran en la consola donde ejecutaste el servidor. Para más detalles, cambia LOG_LEVEL=DEBUG en el archivo .env.

Soporte

Si tienes problemas o preguntas:

  • Consulta los logs del servidor
  • Verifica tu configuración en el archivo .env
  • Asegúrate de tener conexión a internet
